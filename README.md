# Track My Calories
A web app that tracks your calories for the day
by simply entering the foods and the number of calories that those
foods contain.

Viewable on here https://my-calorie-tracker.netlify.app/

# Technologies Used

-HTML & CSS

-JavaScript

# Features

-Fully responsive on all devices

-Unqiue design
